# Horse animation assets

Place your horse video here as **`horse-sprite.mp4`** for the hero section.

The hero also checks **`/public/images/horse-sprite.mp4`** if the animations folder file is missing.

Video requirements for best results:
- MP4 (H.264), loop-friendly clip
- Transparent or dark background works best on the chocolate hero
- Recommended size: under 5MB for fast loading (compress if needed)
